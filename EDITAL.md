## 📖 Descrição

Este projeto faz parte da Unidade Curricular **Programação de Soluções Computacionais** e tem como objetivo o desenvolvimento de um sistema completo.

A nota obtida neste projeto será utilizada na composição da **A3**.

## 💻 Requisitos do Sistema

### 🖥️ Plataforma
- Aplicação **desktop**
- Interface gráfica
- Integração com banco de dados

### 🎨 Interface
- Desenvolvida com **Java Swing (`javax.swing`)**

### 🗄️ Banco de Dados
- Gerenciado com **MySQL**

---

## ⚙️ Funcionalidades Obrigatórias

O sistema deve conter:

### 🔐 Controle de Acesso
- Apenas usuários autenticados podem acessar o sistema

### 👤 Tipos de Usuário
- No mínimo **2 perfis**, por exemplo:
  - Administrador (acesso total)
  - Usuário comum (acesso limitado)

### 🔄 CRUDs
- No mínimo **3 operações CRUD**:
  - Create (Criar)
  - Read (Ler)
  - Update (Atualizar)
  - Delete (Excluir)

### 📊 Dashboard
- Tela com métricas, como:
  - Quantidade de usuários cadastrados
  - Dados relevantes do sistema

### 👑 Usuário Administrador
- Deve ser criado **diretamente no banco de dados**
