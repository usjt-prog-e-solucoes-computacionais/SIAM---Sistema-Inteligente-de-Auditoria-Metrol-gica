## 🚀 Guia Rápido para o Time

1. **Sempre comece atualizando sua Develop:**
   `git checkout dev` -> `git pull origin dev`

2. **Crie uma branch para sua tarefa:**
   `git checkout -b feature/nome-da-tarefa`

3. **Suba suas alterações:**
   `git add .` -> `git commit -m "feat: descrição da tarefa"` -> `git push origin feature/nome-da-tarefa`

4. **Abra um Pull Request (PR):**
   No GitHub, solicite o merge da sua `feature` para a `dev`. Aguarde a revisão do Danilo.
